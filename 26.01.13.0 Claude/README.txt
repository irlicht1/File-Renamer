✨ ГЛАВНАЯ ОСОБЕННОСТЬ - СУФФИКС _AS:
═══════════════════════════════════════════════════════════════════

Все файлы получают суффикс _AS перед расширением:

   Было:           Стало:
   video.mp4    →  video_AS.mp4
   document.pdf →  document_AS.pdf
   photo.jpg    →  photo_AS.jpg

При дубликатах:
   video.mp4    →  video_AS.mp4
   video.avi    →  video (1)_AS.avi

❗ ВАЖНО: В таблице НЕТ строки заголовка!

Правильно:              Неправильно:
┌──────────┐           ┌──────────┐
│ video    │ ← A1      │ Имя      │ ← заголовок
│ document │ ← A2      │ video    │
│ photo    │ ← A3      │ document │
└──────────┘           └──────────┘

Поддерживаемые форматы:
   • Excel (.xlsx, .xls)
   • CSV (.csv)

Используется только ПЕРВЫЙ СТОЛБЕЦ (A).
Пустые строки автоматически пропускаются.

🔄 КАК РАБОТАЕТ ПРОГРАММА:
═══════════════════════════════════════════════════════════════════

1. Файлы в папке сортируются ПО АЛФАВИТУ (A-Z, А-Я)

   Пример папки:
   • 1.mp4
   • 2.pdf
   • 3.jpg
   • 4.png

2. Берутся имена из таблицы (с первой строки):

   A1: video
   A2: document
   A3: video      ← дубликат
   A4: photo

3. Файлы переименовываются по порядку:

   1.mp4 → video_AS.mp4
   2.pdf → document_AS.pdf
   3.jpg → video (1)_AS.jpg    ← дубликат получает номер
   4.png → photo_AS.png

🔧 РЕШЕНИЕ ПРОБЛЕМ:
═══════════════════════════════════════════════════════════════════

❌ Проблема: "Python не найден"
   ✅ Решение:
      1. Скачайте с https://www.python.org/downloads/
      2. При установке поставьте "Add Python to PATH" ✓
      3. Перезагрузите компьютер
      4. Запустите УСТАНОВКА.bat

❌ Проблема: "No module named 'pandas'"
   ✅ Решение:
      Запустите УСТАНОВКА.bat

❌ Проблема: Не запускается двойным кликом на .pyw
   ✅ Решение:
      1. Используйте ПУСК.bat
      2. Или Запуск.vbs
      3. Проверьте установку Python

❌ Проблема: "Permission denied"
   ✅ Решение:
      1. Закройте все файлы в папке
      2. Закройте программы, использующие файлы
      3. Запустите от имени администратора

❌ Проблема: Неправильные имена после переименования
   ✅ Решение:
      1. Проверьте, что в таблице НЕТ строки заголовка
      2. Используйте ПРЕДПРОСМОТР для проверки
      3. Данные должны начинаться с ячейки A1
