# File-Renamer
Версии сборок автоматического переименователя файлов

25.12.1

Требует подстановки в первый столбец электронной таблицы excel список всех файлов в папке для переименования

25.12.1.2

Добавлена функция принудительного переименования расширения файлов в .mp4 - независимо от исходника


_______

Требования к новой версии

* Необходима только одна колонка в таблице excel
* Необходимо добавлять копию файла с тем же именем и расширением путем добавление цифры в конец (в скобках) если время создания между двумя файлами меньше или равно 8 минутам
* Проблема: что делать с дублями?
